# ✅ Todo Manager

A JavaScript practice project for managing tasks, marking them as complete, and organizing work in a simple way.

## ✨ What this project does

This project helps you practice working with:

- arrays of objects
- task status updates
- filtering completed and pending tasks
- counting tasks
- basic real-world problem solving with JavaScript

## 📥 Inputs

The main input is an array of task objects.
Each task object contains:

- `title`: the name of the task
- `completed`: `true` or `false`

Example:

```js
const tasks = [
  { title: "Study JavaScript", completed: false },
  { title: "Exercise", completed: true },
];
```

## 📤 Outputs

The project should be able to:

- add a new task
- mark a task as completed
- show pending tasks
- show completed tasks
- count total, completed, and pending tasks

## ▶️ How to run

1. Open this folder in VS Code.
2. Run the file with Node.js:

```bash
node todo_manager.js
```
